# drf-restfull-employee-service
A simple REST APIs to manage employee service build with Django Rest Framework
# Features:
- get a list of employees or get by id
- perform search for employees
- do bulk creation
- update based on id
- delete an employee by id
# Run:
- git clone <>
- pip install -r requirements.txt
- python manage.py makemigrations
- python manage.py migrate
- python manage.py runserver

  Note: Please refer to the API doc and steps.txt
